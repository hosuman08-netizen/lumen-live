# Lumen Live — Creator Rooms (demo)

A client-only demo of a live social platform for creators. Go live, host a room,
grow a community, and keep a session journal. No backend, no real payments, no
real people — everything runs in the browser with localStorage.

**Core ideas**
- **Sign in & Sparks**: a simple demo account and an in-app credit called Sparks used for
  room entry, tips, and passes. Sparks are fictional demo credits with no real-world value.
- **Live rooms**: creators go live with a mic check; viewers spend Sparks to join and watch.
- **Communities**: member-gated group chats with a Sparks pass.
- **Live Energy Meter**: a friendly meter that reacts to real mic input and room activity.
- **Session Journal**: post-session notes so creators remember what worked.

**Design**
- Warm, low-contrast dark theme with a single gold accent.
- One clear primary action per screen ("Go Live").
- Honest numbers: seats, pot, and odds shown in the UI match what the code does.

**Disclaimer**: This is a fictional demo for 18+ audiences. It depicts no real people
and processes no real money or payments.

**Roadmap (future)**: real accounts, live video, and a real backend. This build is a
front-end prototype only.
